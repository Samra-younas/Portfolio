## 🚀 Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Samra-younas/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Set up environment:**
   ```bash
   cp .env.example .env
   # Add your ANTHROPIC_API_KEY to .env file
   ```

4. **Run the application:**
   ```bash
   python src/app.py
   ```

5. **Access the portfolio:**
   - Local: http://localhost:5000
   - Live: https://samra.pythonanywhere.com
